Project Page

Model Railroad Dispatcher

A custom built dispatching software system built for controlling model railroads with low-cost hardware. This Java-written program allows modeling of realistic complex railroad operations. With a variety of functions and control features, I am taking my hobby to the next level by developing a system that can handle and optimize a miniature railway network under the control of a single person.

Model Railroad Hardware

This project details the development process of integrating Arduino-based hardware into a model railroad. It includes an array of devices such as microcontrollers, servos, relays, infrared sensors, and custom signal control boards.

Model Railroad Assets

This project is dedicated to producing a series of detailed 3D models that will enhance model railroad layouts. It includes the creation of track infrastructure components, buildings, and scenic elements, all designed with accuracy and attention to detail. The goal is to provide model railroaders with realistic and functional assets that integrate seamlessly into existing setups.
